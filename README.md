# Cosas a tener en cuenta del proyecto
1. Conteo de palabras y números:
El formato según el manual de estilo Normas APA 7, existe una regla estricta sobre cuándo escribir el número con dígitos y cuándo con letras: Números del 0 al 9: Se deben escribir con letras (ej. tres experimentos, nueve participantes). Esto sumará tantas palabras como uses.  Números del 10 en adelante: Se deben escribir con cifras (ej. 48 perros, 150 muestras). El procesador lo contará como una única palabra. 
2. Sabemos que los espacios y saltos de línea se cuentan
